# Elevator (Lift) Management System

## Description

This is a simple Elevator (Lift) Management System that simulates the operation of lifts in a building. It supports multiple floors, lift requests, door management, and more.

## Installation

- Clone this repository to your local machine.
- Compile and run the code using a C++ compiler.

## Usage

- Enter a floor number (0-5) to request a lift.
- Enter '0' to exit the program.
- Enter '-1' to close the lift doors (if they are open).

## Features

- Support for 5 floors, including a ground floor.
- Requesting a lift to a specific floor.
- Opening and closing lift doors.
- Exiting the program.
